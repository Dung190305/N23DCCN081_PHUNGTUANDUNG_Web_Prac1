# Bài thực hành 1: Blog Listing Page

**Họ tên:** Trần Thị B  
**MSSV:** 2024056  
**Lớp:** Lập trình Web - Nhóm 2

## Mô tả
Dự án xây dựng trang danh sách blog và trang chi tiết sử dụng NextJS App Router và Tailwind CSS. Dữ liệu được lấy từ JSONPlaceholder API.

## Công nghệ sử dụng
- Next.js 15 (App Router)
- Tailwind CSS
- Git & GitHub

## Cách chạy
1. Clone repo
2. `npm install`
3. `npm run dev`
